# 介绍

Typography组件 的 vue 实现。

# 特点

  • 易于使用.
  • 同时适用于 Vue3 和 Vue2。( 感谢vue-demi ).
  • 由 TS 编写，完整的类型推导.

# 安装

yarn add @typography/vue

# 使用

import { Paragraph } from '@typography/vue';
